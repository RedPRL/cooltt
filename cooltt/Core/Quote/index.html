Module Core.Quote

The purpose of this module is to transform semantic objects into syntactic objects as efficiently as possible; only user-specified top-level definitions will be unfolded, in accordance with the veil (see Monads.QuM.read_veil).

module D := Domain
module S := Syntax

val quote_tp : D.tp -> S.tp Monads.quote
val quote_cut : D.cut -> S.t Monads.quote
val quote_cof : D.cof -> S.t Monads.quote
val quote_dim : D.dim -> S.t Monads.quote
